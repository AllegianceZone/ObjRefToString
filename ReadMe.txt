========================================================================
    CONSOLE APPLICATION : ObjRef2String Project Overview
========================================================================

Imago has created this ObjRef2String application for Allegiance on 8/10.

This file contains a summary of what you will find in each of the files that
make up your ObjRef2String application and what the application does.

ObjRef2String.vcxproj
    This is the main project file for ObjRef2String generated using hours of sweat and tears.
    It contains information about the version of Visual C++ that generated the file, and
    information about the platforms, configurations, and project features used.

ObjRef2String.vcxproj.filters
    This is the filters file for VC++ projects generated using an Application Wizard. 

ObjRef2String.cpp
    This is the main application source file.

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named ObjRef2String.pch and a precompiled types file named StdAfx.obj.

/////////////////////////////////////////////////////////////////////////////
Other notes:

ObjRef2String uses Zlib and Test projects from the Allegiance source, 
this project should just be added to your Allegiance.sln.
